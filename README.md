# sevrer-for-clariti


## Getting Started
This is for a supabase backend for another project

```bash
# Clone the repo
git clone https://github.com/Zekepeke/sevrer-for-clariti.git 
cd server-for-clariti
```

# from server directory
python3 -m venv .venv
# macOS/Linux
source .venv/bin/activate

# running 
uvicorn myapi:app --reload
